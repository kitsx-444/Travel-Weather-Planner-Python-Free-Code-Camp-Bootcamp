# Travel-Weather-Planner-Python-Free-Code-Camp-Bootcamp

What is this program about?
This program is a travel weather planner. It uses boolean values and you change the variables directly in the code to test different scenarios on how many miles of travel ahead of you. It then runs through many conditions and outputs a true or false value. True means travel is possible and false isn't

What did I learn?
The concepts used in this challenge weren't new to me but the way they were used were. What is it I am referring to: learning that you can nest if/else inside elif blocks.

What I struggled with?
1. Understanding why elif block order matters and realizing that if Python reads (from top to bottom) and one block was true it ignored all the rest.
2. Creating ranges.
